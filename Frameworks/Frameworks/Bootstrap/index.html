Bootstrap

Página inicial | Web oficial de Bootstrap

Descripción: Bootstrap es el framework CSS más popular para crear sitios web responsivos y modernos con facilidad.

  • Ventajas: Gran cantidad de componentes predefinidos, fácil de aprender, excelente documentación y soporte comunitario.
  • Inconvenientes: Personalización limitada sin sobrescribir estilos, puede hacer que los sitios se vean similares.

Instalación

Instalación: Tendremos que poner estos 2 comandos en el HTML para que funcione el framework.

1. <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

2. <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

Link instalación Bootstrap

Componentes comunes

1. Botón: sirve para ejecutar una acción cuando el usuario hace clic, como enviar un formulario, abrir un enlace o activar una función.

2. Formulario: se utiliza para recoger información del usuario, como nombres, correos, contraseñas o comentarios, y enviarla al servidor.

3. Tarjeta (Card): se usa para mostrar contenido agrupado visualmente, como texto, imágenes, precios o descripciones, dentro de un contenedor con borde o sombra.

Componenetes comunes: Tendremos que copiar y pegar estos comandos en el HTML para que se ejecuten.

  • Botón: <button class="btn btn-primary">Enviar</button>
  • Formulario: <form><input class="form-control" type="text" placeholder="Nombre"></form>
  • Tarjeta: <div class="card"><div class="card-body">Contenido</div></div>
