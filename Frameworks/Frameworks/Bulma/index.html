Bulma

Página inicial | Web oficial de Bulma

Descripción: Bulma es un framework CSS moderno y ligero basado en flexbox, que facilita la creación de interfaces responsivas sin necesidad de JavaScript.

  • Ventajas: Fácil de aprender y usar, completamente basado en CSS, muy flexible y ligero.
  • Inconvenientes: Menor cantidad de componentes en comparación con otros frameworks, algunos componentes avanzados requieren desarrollo extra.

Instalación

Instalación: Tendremos que poner este comando en el HTML para que funcione el framework.

1. <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bulma@0.9.4/css/bulma.min.css">

Link instalación Bulma

Componentes comunes

1. Botón: sirve para ejecutar una acción cuando el usuario hace clic, como enviar un formulario, abrir un enlace o activar una función.

2. Formulario: se utiliza para recoger información del usuario, como nombres, correos, contraseñas o comentarios, y enviarla al servidor.

3. Tarjeta (Card): se usa para mostrar contenido agrupado visualmente, como texto, imágenes, precios o descripciones, dentro de un contenedor con borde o sombra.

Componenetes comunes: Tendremos que copiar y pegar estos comandos en el HTML para que se ejecuten.

  • Botón: <button class="button is-primary">Enviar</button>
  • Formulario: <form><input class="input" type="text" placeholder="Nombre"></form>
  • Tarjeta: <div class="card"><div class="card-content">Contenido</div></div>
