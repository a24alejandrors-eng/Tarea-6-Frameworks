Tailwind CSS

Página inicial | Sitio oficial de Tailwind CSS

Descripción: Tailwind CSS es un framework de utilidades que permite un diseño altamente personalizable y rápido mediante clases.

  • Ventajas: Personalización total, código limpio y modular, ideal para proyectos únicos.
  • Inconvenientes: Curva de aprendizaje inicial, puede provocar HTML con muchas clases.

Instalación

Instalación: Tendremos que poner este comando en el HTML para que funcione el framework.

1. <script src="https://cdn.tailwindcss.com"></script>

Link instalación Tailwind CSS

Componentes comunes

1. Botón: sirve para ejecutar una acción cuando el usuario hace clic, como enviar un formulario, abrir un enlace o activar una función.

2. Formulario: se utiliza para recoger información del usuario, como nombres, correos, contraseñas o comentarios, y enviarla al servidor.

3. Tarjeta (Card): se usa para mostrar contenido agrupado visualmente, como texto, imágenes, precios o descripciones, dentro de un contenedor con borde o sombra.

Componenetes comunes: Tendremos que copiar y pegar estos comandos en el HTML para que se ejecuten.

  • Botón: <button class="bg-blue-500 text-white px-4 py-2 rounded">Enviar</button>
  • Formulario: <form><input class="border rounded px-2 py-1" type="text" placeholder="Nombre"></form>
  • Tarjeta: <div class="bg-white shadow rounded p-4">Contenido</div>
