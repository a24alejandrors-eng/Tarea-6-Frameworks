Materialize

Página inicial | Web oficial de Materialize

Descripción: Materialize es un framework CSS basado en Material Design de Google, ideal para crear interfaces modernas y responsivas rápidamente.

  • Ventajas: Diseño atractivo y moderno, componentes interactivos listos para usar, fácil integración con proyectos nuevos.
  • Inconvenientes: Menor flexibilidad en personalización, tamaño relativamente grande, menos popular que Bootstrap.

Instalación

Instalación: Tendremos que poner estos 2 comandos en el HTML para que funcione el framework.

1. <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/css/materialize.min.css">

2. <script src="https://cdnjs.cloudflare.com/ajax/libs/materialize/1.0.0/js/materialize.min.js"></script>

Link instalación Materialize

Componentes comunes

1. Botón: sirve para ejecutar una acción cuando el usuario hace clic, como enviar un formulario, abrir un enlace o activar una función.

2. Formulario: se utiliza para recoger información del usuario, como nombres, correos, contraseñas o comentarios, y enviarla al servidor.

3. Tarjeta (Card): se usa para mostrar contenido agrupado visualmente, como texto, imágenes, precios o descripciones, dentro de un contenedor con borde o sombra.

Componenetes comunes: Tendremos que copiar y pegar estos comandos en el HTML para que se ejecuten.

  • Botón: <a class="btn waves-effect waves-light">Enviar</a>
  • Formulario: <div class="input-field"><input id="nombre" type="text"><label for="nombre">Nombre</label></div>
  • Tarjeta: <div class="card"><div class="card-content">Contenido</div></div>
